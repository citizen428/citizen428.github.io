On ruby-talk somebody "was asking":http://www.ruby-forum.com/topic/190552 for a method which will be used to generate a random string, by returning either a lowercase letter or one of the digits 0-9 when called. As an example he posted the following Pascal program:

{% codeblock lang:pascal %}
function GetRandomChar: char;
var
 r: integer;
begin
 r := random(36);
 case r of
   0..25: result := chr(ord('a') + r);
   else : result := chr(ord('0') + r);
 end;
end;
{% endcodeblock %}
