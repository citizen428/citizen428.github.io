I love all the little discussions on RubyLearning, especially when they lead to me writing stupid code the world doesn't need. ;-) Today: a poor man's <a href="http://en.wikipedia.org/wiki/Read-eval-print_loop">REPL</a> in 2 lines of Ruby 1.9:
{% codeblock lang:ruby %}
repl = -> prompt { print prompt; puts(" => %s" % eval(gets.chomp!)) }
loop { repl[">> "] }
{% endcodeblock %}
Behold the wonder:
{% codeblock lang:ruby %}
>> 2 + 3
 => 5
>> "foo".upcase
 => FOO
>> [].class
 => Array
>> %w(foo bar)[0]
 => foo
{% endcodeblock %}
